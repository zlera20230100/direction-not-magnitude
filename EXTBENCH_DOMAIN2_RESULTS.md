# External benchmark #2 — gradient-reliability gate on a DIFFUSION (heat-conduction) BVP

**Goal.** Test whether the deep-ensemble *sign-agreement* gate (Paper 2) generalises to a
**structurally different physics family**, beyond the resonant cases already in the manuscript:
the data-free EM-PINN antenna (resonant Maxwell) and the external thin-film TMM bench
(Fabry–Perot resonance). This second external benchmark is **elliptic diffusion**, not resonance.

Script: `extbench_poisson.py` → data: `extbench_poisson.npz`.
Reuses verbatim from `extbench_tmm.py`: tiny MLP surrogate (K→128→128→128→1, SiLU), deep ensemble
(M=10 fixed seeds, identical training recipe), per-component sign-agreement gate
`sa = max(frac(g>0), frac(g<0))`, pooled AUC of "sign-agreement predicts whether each autodiff
gradient component is sign-correct vs the oracle", and the least-trusted-first allocator
(cosine-to-truth vs #oracle-solves for gate / random / oracle ordering). The **only** thing changed
is the physics. Run on CPU (`D:\ANACONDA\python.exe`, `OMP_NUM_THREADS=6`).

## Chosen problem

**1-D steady-state heat conduction (inverse conductivity design).** Fourier conduction on x∈[0,1]
with a fixed internal source q(x) and Dirichlet ends:

  −d/dx( g(x) dT/dx ) = q(x),  T(0)=1, T(1)=0.

- **Design vector:** per-segment conductivity **g ∈ ℝ^K** (g_k > 0), K segments along the bar →
  a genuine multi-component design gradient.
- **Oracle (exact, cheap):** finite-volume discretisation with harmonic-mean face conductivity on
  a fine grid (NGRID=160) → SPD **tridiagonal** system → Thomas algorithm. One "solve" = one BVP
  solve, the analogue of one TMM / full-wave evaluation.
- **Objective (minimised):** profile-matching MSE J(g) = mean_i (T_i(g) − T_target,i)²; the target
  field is generated by a known high-contrast conductivity profile, so a perfect design exists
  (J(G_TRUE)=0).
- **Ground-truth gradient:** central finite difference of J through the oracle BVP solve (2K solves).

This is a self-adjoint, positive-definite, **non-oscillatory** operator — the opposite character to
a Fabry–Perot flank — so it is a real cross-family test of the gate.

## Two regimes (honesty first)

A smooth, well-conditioned conduction design is **too easy** for this test: the cheap MLP already
gets ~100 % of gradient-component *signs* right, so there are **no** sign-wrong components, the gate
AUC is **undefined**, and the allocator has nothing to save. This is an honest finding in its own
right: *for well-conditioned diffusion design, a cheap surrogate already yields reliable gradient
signs and the gate is simply not needed.* The script logs this "easy" regime for the record.

The **main** benchmark therefore deliberately stresses the problem into a *discriminating* regime
— exactly as the companion `extbench_tmm_hard.py` does for photonics — so that a non-trivial
fraction of components are sign-wrong and the gate/allocator can actually be scored: high-contrast
(log-normal) conductivities, a **sharp sign-changing** internal source (heater + sink), **K=16**
segments (each segment's influence is small / sign-fragile), **few** training samples (NS=70), and
query points roaming to **±25 %** (mild extrapolation).

## Results — MAIN (discriminating) regime

Pooled over NQ=100 query points × K=16 components = **1600** gradient components.

| Metric | Value |
|---|---|
| Class balance | **1429 sign-correct / 171 sign-wrong** (10.7 % sign-wrong) |
| **Gate AUC** (sign-agreement → component sign-correct) | **0.761** |
| Bootstrap 95 % CI (NB=5000, same recipe as `auc_ci_device_external.py`) | **[0.720, 0.802]** |
| Mean sign-agreement on sign-CORRECT components | 0.944 |
| Mean sign-agreement on sign-WRONG components | 0.769 |
| Mean # components trusted at τ=0.9 | 12.8 / 16 |

**Allocator — oracle BVP solves to reach a target descent-direction cosine** (base cosine of the
pure surrogate gradient = **0.951**; all-FD baseline = 16 solves):

| target cosine | gate-order | random-order | saved vs random |
|---|---|---|---|
| 0.90 | 0 | 0 | 0 |
| 0.95 | 0 | 0 | 0 |
| 0.99 | **12** | **13** | **1 (8 %)** |

## Results — EASY (well-conditioned) regime, for the record

K=8, NS=400, smooth source, ±10 % roam: **0 / 480 sign-wrong → AUC undefined (nan)**, base
cosine = **1.000**, allocator saves nothing. Confirms the gate is unnecessary when diffusion
gradients are already saturated-correct.

## Honest verdict — does the gate generalise to this domain?

**Partly. The *ranking* signal generalises; the *allocator payoff* does not (in diffusion).**

1. **AUC generalises (above chance, with a tight, well-defined CI).** The sign-agreement gate
   predicts which autodiff gradient components are sign-correct with **AUC 0.761, 95 % CI
   [0.720, 0.802]** — lower bound well above chance (0.5), on a healthy 171-negative class. Mean
   sign-agreement is clearly separated (0.944 correct vs 0.769 wrong). So the *core claim of Paper 2*
   — sign-agreement carries real information about per-component gradient trust — **holds in a
   non-resonant, diffusion physics family.**

2. **But the gate is WEAKER here than in the resonance domain.** The thin-film TMM bench reports
   AUC **0.906 [0.833, 0.965]**; diffusion gives **0.761 [0.720, 0.802]** — a genuinely lower
   (non-overlapping point estimates) discriminative power. We do not hide this: the gate generalises
   in *direction* but not in *strength*.

3. **The allocator saving is small in diffusion** (≤1 solve / 8 % at cosine 0.99, 0 at lower
   targets). The honest mechanism: in conduction the sign-WRONG components are the **low-magnitude**
   ones (segments far from the source / near-zero influence), so flipping their signs barely moves
   the descent-direction cosine — the base surrogate cosine is already 0.951. By contrast, on the
   resonance flank the sign-fragile components can also be high-leverage, so gate-ordered FD
   correction buys a larger cosine gain per solve. **The allocator's benefit is therefore physics-
   dependent**, and Paper 2's "fewer solves" claim should be scoped to problems where unreliable
   components are also high-leverage (resonant/ill-conditioned regimes), not asserted as universal.

**Bottom line for the manuscript:** the gradient-reliability gate is *cross-domain* in the sense
that sign-agreement remains a statistically valid reliability indicator in a diffusion/elasticity-
style physics (AUC ≈ 0.76, CI excludes chance), strengthening the generality claim beyond the two
resonant cases. The companion message — required for honesty — is that **its magnitude (AUC) and
its solver-saving payoff are both smaller in well-conditioned diffusion**, and vanish entirely when
the surrogate's gradient signs are already saturated-correct. This is a *bounded* generalisation,
not a universal one.
